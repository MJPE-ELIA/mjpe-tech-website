# MJPE Tech Website

## 🚀 Présentation

MJPE Tech est une organisation technologique fondée par Elia Mushamuka Jean-Paul (MJPE), visant à développer des solutions numériques innovantes et accessibles en RDC, en Afrique et dans le monde.

## 🎯 Slogan

Transformer la passion en innovation

## 🌍 Objectif du projet

Ce site officiel a pour but de :

- Présenter MJPE Tech
- Partager les nouveautés
- Montrer la vision et la mission
- Permettre le contact avec l'organisation

- HTML5
- CSS3
- JavaScript (Vanilla)
